# ivrcscripts[![Build Status](https://travis-ci.com/bit-ivrc/ivrcscripts.svg?token=Jmj6MSYSGZmX9ePjdawa&branch=master)](https://travis-ci.com/bit-ivrc/ivrcscripts)
Various useful scripts for
* Ubuntu 16.04 LTS
* ROS Kinetic.


## Usage

```
./<*>.sh
```
